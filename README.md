## Growtopia-Proxy
Growtopia proxy that allows modification and viewing of packets, and creation of new ones.

## Supported Platforms
- Termux Android
- Linux

## Build For Linux

```bash
sudo apt update
sudo apt-get install build-essential
sudo apt install gcc
sudo apt-get install libssl-dev
chmod +x LinuxBuild.sh
sudo ./LinuxBuild.sh
```
