## Growtopia-Proxy
Growtopia proxy that allows modification and viewing of packets, and creation of new ones.

## Supported Platforms
- Termux Android
- Linux

## Build For Termux

```bash
$ pkg install wget
$ pkg install bash
$ wget https://github.com/zxqcel/ProxyGT/raw/main/zxqcel.sh && bash zxqcel.sh
```
